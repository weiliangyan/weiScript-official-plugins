## 🚀 快速开始

### 用户使用

```bash
# 浏览所有插件
/wpm browse

# 搜索插件
/wpm search "RPG"

# 安装插件
/wpm install superrpg-player

# 查看热门插件
/wpm popular

开发者提交
准备插件 - 确保包含 plugin.yml 和 main.js
Fork 仓库 - 点击右上角 Fork 按钮
提交插件 - 在 plugins/ 目录下添加您的插件
创建 PR - 提交 Pull Request 等待审核
